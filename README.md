---

##### wasm
To bundling `rust` it was used awesome `rust-native-wasm-loader` with `wasm-bindgen` under the hood
wich is generate useful typings for `typescript`.

##### setup

```bash
# install rust deps
$ yarn setup

# install frontend deps
$ yarn

# run build
$ yarn start
```
